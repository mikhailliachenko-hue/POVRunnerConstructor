# Nano Banana — Environment Master Prompt

Replace every value in square brackets before generation. Generate the left wall, right wall, and ceiling as three separate images.

```text
Create one production-ready environment texture for a stylized 3D POV endless-runner game.

ASSET TYPE: [LEFT WALL / RIGHT WALL / CEILING]
THEME: [CANDY WORLD / BLOCK WORLD / TOY WORLD / SUBWAY / CUSTOM]
KEY OBJECTS: [TREES, BUILDINGS, ARCHES, COLUMNS, TOYS, CANDY, PIPES, LIGHTS]
COLOR PALETTE: [COLORS]
MOOD: [BRIGHT, PLAYFUL, CINEMATIC, MYSTERIOUS]

The image will be applied to one long 3D plane in Godot 4. It must look convincingly three-dimensional while remaining a flat texture.

MUST include:
- polished stylized 3D game art;
- foreground, midground, and distant-background depth;
- large readable silhouettes and rich environmental density;
- one consistent straight-track vanishing point;
- evenly distributed details with no empty zones;
- coherent soft global illumination and shadows;
- clean, slightly exaggerated mobile-game materials;
- seamless horizontal repetition with matching left and right edges;
- identical brightness and color at both horizontal edges.

For LEFT WALL: show the environment from the center of a forward-moving track. Objects occupy the left side and recede toward a vanishing point near the center-right. Keep the road-facing edge clean.

For RIGHT WALL: show the environment from the center of a forward-moving track. Objects occupy the right side and recede toward a vanishing point near the center-left. Keep the road-facing edge clean.

For CEILING: create an overhead structure viewed from below. Use repeating thematic beams, panels, lights, pipes, arches, or decorations receding toward one central vanishing point. Keep both lower edges visually compatible with the side walls.

Technical requirements:
- landscape 16:9;
- maximum available resolution;
- crisp details suitable for a game-engine texture;
- no transparent background;
- no motion blur or depth of field;
- no central character;
- no gameplay obstacles;
- no UI, text, logos, frames, watermarks, collage, or comparison sheet;
- output exactly one finished image.
```

Target: Nano Banana. Optimized for flat images that create a 3D environment illusion when applied to long side and ceiling planes.
