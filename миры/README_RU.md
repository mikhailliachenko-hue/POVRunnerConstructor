# Импортированные миры

Создайте внутри этой папки отдельную папку мира и положите в неё один файл
`.glb` или `.gltf`. Например: `миры/my_world/world.glb`.

Конструктор автоматически найдёт мир и создаст рядом `world_config.cfg` после
первого запуска. Исходная 3D-модель не изменяется и не перезаписывается.
