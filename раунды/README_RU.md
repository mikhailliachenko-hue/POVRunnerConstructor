# Наборы раундов

Каждый `round_XX` является отдельным набором моделей и окружения. Раскладывайте GLB/GLTF по папкам `jump`, `duck`, `left`, `right` и `smash`.

Изображения окружения находятся в `textures`:

- `road` — текстура дороги;
- `ground` — текстура земли вокруг дороги;
- `background` — панорамный задний фон.

В каждой папке используется первое изображение по алфавиту. В конструкторе весь набор выбирается полем `ROUND CONTENT`.
