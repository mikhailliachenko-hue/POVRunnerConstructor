# Случайные карточки на дороге

Добавляйте сюда изображения PNG, JPG, JPEG или WEBP. В конструкторе их показ включает переключатель `RANDOM IMAGE PANELS`.

При сборке раунда изображения перемешиваются, появляются в случайных подготовленных местах и не повторяются в пределах одного прохождения. Карточки не имеют коллизии: игрок проходит сквозь них и получает `NICE!`.

Для прозрачного фона используйте PNG или WEBP с альфа-каналом. Соотношение сторон изображения сохраняется автоматически.
