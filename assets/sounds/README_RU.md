# Звуки проекта

Добавляйте файлы WAV, OGG или MP3 в папку нужного события:

- `jump` — прыжок;
- `land` — приземление;
- `dodge` — движение LEFT/RIGHT;
- `duck` — приседание;
- `smash` — удар;
- `nice` — успешное действие;
- `music` — музыка (зарезервировано);
- `ambience` — окружение (зарезервировано).

Если в папке события лежит несколько файлов, игра случайно выбирает один из них. Если папка пуста, используется синтетическая звуковая заглушка.
