SMASH WALL TEXTURE SLOTS

Place two ordinary image files in this folder:

1. intact.png  - the normal, undamaged wall image
2. cracked.png - the same wall image after the first hit, with cracks

Both images should use the same dimensions and composition so the switch is seamless.
Recommended size: 2048x2048 PNG.

The wall is one unified mesh and each image is applied as one unified texture.
