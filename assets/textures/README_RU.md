# Пользовательские текстуры

Скопируйте изображения PNG, JPG, JPEG или WebP в соответствующую папку:

- `road` — поверхность дороги;
- `ground` — земля или пол вокруг дороги;
- `background` — фон окружения;
- `left_wall` — левая стена закрытого коридора;
- `right_wall` — правая стена закрытого коридора;
- `ceiling` — потолок закрытого коридора.

Пока эти каталоги являются подготовленными слотами. Автоматическое применение текстур будет подключено на следующем этапе.
