# Модели препятствия JUMP

Сюда будем складывать визуальные 3D-модели препятствия для прыжка в формате `.glb` или `.gltf`.

Игровая логика и невидимая коллизия JUMP находятся отдельно в коде. Поэтому загруженная сюда модель будет отвечать только за внешний вид: её можно менять, масштабировать или удалять, не ломая механику прыжка.

Пока папка пуста, игра показывает яркий куб-заглушку размером 1,35 × 1,35 × 1,35 метра.
