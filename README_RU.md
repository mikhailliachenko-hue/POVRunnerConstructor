# POV Runner Constructor — прототип 0.1

Это первая рабочая основа конструктора: POV-камера автоматически движется по трассе, а игрок использует три действия.

## Как запустить

1. Установите Godot 4 с официального сайта: https://godotengine.org/download/windows/
2. Запустите Godot.
3. Нажмите **Import**.
4. Выберите файл `project.godot` из этой папки.
5. Откройте проект и нажмите кнопку ▶ или клавишу **F6/F5**.

## Управление

- **A** или **←** — перейти влево.
- **D** или **→** — перейти вправо.
- **Пробел**, **W** или **↑** — прыгнуть.
- **R** — начать трассу заново.
- **Tab** — показать или скрыть панель настроек.

## Что уже работает

- автоматическое движение вперёд;
- три полосы;
- плавные перестроения;
- прыжок;
- шесть тестовых препятствий;
- предупреждения LEFT, RIGHT и JUMP;
- шкала дистанции;
- финиш;
- изменение скорости, высоты прыжка и плавности прямо во время запуска.
